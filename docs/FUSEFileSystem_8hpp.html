Detailed Description

The header file for the FUSEFileSystem class.

Author
Santhosh Ranganathan

This file contains the class definition for the FUSEFileSystem class. The FUSEFileSystem class handles initializing and shutting down of the FUSE filesystem and also communication with the TaggableFS daemon for various filesystem operations.
