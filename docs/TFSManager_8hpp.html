The header file for the TFSManager class.

Author
Santhosh Ranganathan

This file contains the class definition for the TFSManager class. The TFSManager class handles all functionalities related to TaggableFS' filesystem and answers the queries from FUSE operations and the command line queries from the user. It stores and updates files in the root directory named with their hash values and stores the actual filenames in a database. Folders and tags are saved to the same table. The initial idea was have a unified view where tagged files are in a reserved folder and allow tagging by creating folders and moving files in a file manager. In the final submitted version, there are instead two modes: one to copy files into the filesystem and the other to peruse, create and delete tags.
